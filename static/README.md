# Ragent 网页界面使用指南

## 快速启动

### 1. 启动后端服务

在项目根目录运行：

```bash
uvicorn app.main:app --reload --host 127.0.0.1 --port 8001
```

### 2. 访问网页界面

在浏览器中打开：

```
http://127.0.0.1:8001
```

## 功能特性

### 🎨 现代化设计
- 深色主题，紫色渐变配色
- 流畅的动画效果
- 响应式布局，支持移动端

### 💬 智能对话
- 支持流式输出和普通模式
- 实时打字动画
- 自动保存会话历史
- Markdown 格式支持（代码块、粗体、斜体、链接）

### ⚙️ 灵活配置
- 可调整 API 地址
- 自定义检索数量（top_k）
- 启用/禁用查询重写
- 调试模式查看详细信息

### 🚀 快捷操作
- 预设常用问题按钮
- 一键清空对话
- Enter 发送，Shift+Enter 换行

## 界面说明

### 主界面
- **顶部导航栏**：显示 Logo 和操作按钮
- **聊天区域**：显示对话历史
- **输入框**：输入问题，支持多行文本
- **选项开关**：流式输出、调试模式

### 设置面板
点击右上角齿轮图标打开设置：
- **API 地址**：后端服务地址（默认 http://127.0.0.1:8001）
- **检索数量**：RAG 检索的文档数量（1-20）
- **查询重写**：是否启用智能查询优化

## 技术栈

- **前端**：原生 HTML5 + CSS3 + JavaScript
- **后端**：FastAPI + Python
- **特性**：SSE 流式传输、CORS 支持、静态文件服务

## 注意事项

1. 确保后端服务已启动
2. 首次使用建议检查设置中的 API 地址
3. 流式模式提供更好的交互体验
4. 调试模式会显示额外的技术信息

## 故障排查

### 无法连接到服务器
- 检查后端服务是否正常运行
- 确认 API 地址配置正确
- 查看浏览器控制台的错误信息

### 消息发送失败
- 确认网络连接正常
- 检查后端日志是否有错误
- 尝试关闭调试模式重新发送
