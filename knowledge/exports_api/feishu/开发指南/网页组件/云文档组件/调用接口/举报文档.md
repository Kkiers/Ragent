# 举报文档
## 示例
```js
// npm
myComponent.invoke(DocComponentEvent.REPORT_ABUSE).then(function(response) {
  const { code, msg } = response;
  // ...
});

// sdk
myComponent.invoke('REPORT_ABUSE').then(function(response) {
  const { code, msg } = response;
  // ...
});
```

## 返回
|属性|	类型|	说明|
| ---|----- | ------- | 
|code|	Number |	0：成功 、 -1：失败 |
|msg|	String |	信息 |
