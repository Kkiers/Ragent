# 点击评论划线时

点击正文中评论划线内容时触发该事件。
## 示例
```js
// npm
myComponent.register(DocComponentEvent.ON_ACTIVE_COMMENT, function(eventData) {
   const {
       commentId // 真实评论 id
   } = eventData;
});

// sdk
myComponent.register('ON_ACTIVE_COMMENT', function(eventData) {
   const {
       commentId // 真实评论 id
   } = eventData;
});
```

## 返回
|属性|	类型|	说明|
| ---|----- | ------- | 
|commentId|String |评论 id 或临时评论id
