# 文档目录变化
## 示例
```js
myComponent.event.onDirectoryChange(function(directoryEntryList) {
  // ...
});
```

## 返回
|属性|	类型|	说明|
| ---|----- | ------- | 
|directoryEntryList|	Array |文档目录列表

### directoryEntry
|属性|	类型|	说明|
| ---|----- | ------ | 
|anchor|	String |	锚点 hash 值 （初始化时获取的数据不包含此项）
|text|	String| 目录标题
|indentLevel|	Number|	缩进等级|
