# padding-bottom

## 介绍

指定一个元素内容与下边界的距离，该属性不能为负值。

## 语法

```css
/* <length> values */

padding-bottom: 0.5em;

padding-bottom: 0;

padding-bottom: 2px;

/* <percentage> value */

padding-bottom: 10%;
```

### 取值

-   `<length>`

以固定值为内边距。

-   `<percentage>`

相对于包含块的宽度，以百分比值为内边距。

### 标准化语法

```css
padding-bottom: <length> | <percentage>
```
