# 创建全场满额送活动

路径：产品文档/运营工具/支付有礼/API列表/支付有礼活动/创建全场满额送活动

使用场景：商户可以创建满额送活动，用户支付后送全场券，提升交易额。
可调用商户：商户/服务商/渠道商

## 接口说明

支持商户：【普通商户】

请求方式：【POST】`/v3/marketing/paygiftactivity/unique-threshold-activity`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点 ，指引[点击查看](https://pay.weixin.qq.com/doc/v3/merchant/4012075113)
