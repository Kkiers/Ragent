# 下载电子回单

路径：产品文档/运营工具/商家转账/API列表/获取电子回单/下载电子回单

通过该接口，商户可以下载商家转账业务的电子回单。

- 下载电子回单后，需将实际电子回单的哈希值与从接口返回的哈希值进行比对，以确保数据的完整性。

## 接口说明

支持商户： 【普通商户】

请求URL： 通过查询电子回单接口返回参数`download_url`获取URL链接，链接地址10分钟内有效。

请求方式： GET

前置条件： 通过查询电子回单接口返回参数`download_url`获取URL链接。

## 请求步骤说明

1. 先通过查询电子回单接口获取download_url
2. 获取到电子回单文件资源的下载地址信息后，参考下方示例。
3. 通过微信支付API v3标准对通过查单获取到的download_url进行签名，并发起请求，即可下载到电子回单文件的数据流。参考下方示例。

### 回单文件资源的下载地址示例
![](https://gtimg.wechatpay.cn/resource/xres/img/202409/5236a3a065efe1276d29855d7cad26f8_16x16.svg)
```
1{
2 "hash_type": "SHA1",
3 "hash_value": "79bb0f45fc4c42234a918000b2668d689e2bde04",
4 "download_url": " https://api.mch.weixin.qq.com/v3/transferdownload/signfile?token=xxx"
5}
```
