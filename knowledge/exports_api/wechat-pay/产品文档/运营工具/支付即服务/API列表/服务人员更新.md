# 服务人员更新

路径：产品文档/运营工具/支付即服务/API列表/服务人员更新

用于为商户更新门店服务人员的姓名、头像等信息。

注意：

个人微信商家、企业微信商家均支持服务人员信息更新

## 接口说明

支持商户：【普通商户】

请求方式：【PATCH】`/v3/smartguide/guides/{guide_id}`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点 ，指引[点击查看](https://pay.weixin.qq.com/doc/v3/merchant/4012075113)

## 请求参数

Header   HTTP头参数

 Authorization 　必填　string

请参考[签名认证](https://pay.weixin.qq.com/doc/v3/merchant/4012365342#%E8%AF%B7%E6%B1%82%E6%8E%A5%E5%8F%A3%E5%90%8E%E7%AB%AF%E7%AD%BE%E5%90%8D)生成认证信息

 Accept 　必填　string

请设置为`application/json`

 Content-Type 　必填　string

请设置为`application/json`

 Wechatpay-Serial 　必填　string

【微信支付公钥ID】或【微信支付平台证书序列号】请求参数中的敏感字段，需要使用微信支付公钥加密（推荐），请参考[获取微信支付公钥ID说明](https://pay.weixin.qq.com/doc/v3/merchant/4013038816#Q%3A-%E5%A6%82%E4%BD%95%E8%8E%B7%E5%8F%96%E5%BE%AE%E4%BF%A1%E6%94%AF%E4%BB%98%E5%85%AC%E9%92%A5ID%3F)以及[微信支付公钥加密敏感信息指引](https://pay.weixin.qq.com/doc/v3/merchant/4013053257)，也可以使用微信支付平台证书公钥加密，参考[获取平台证书序列号](https://pay.weixin.qq.com/doc/v3/merchant/4013053294)、[平台证书加密敏感信息指引](https://pay.weixin.qq.com/doc/v3/merchant/4013053264)

path  路径参数

 guide_id 　必填   string(64)

**【服务人员ID】**服务人员在支付即服务系统中的唯一标识

body  包体参数

 name 　选填   string

【服务人员姓名】需更新的服务人员姓名，不更新不需传入，该字段需要使用微信支付公钥加密（推荐），请参考[获取微信支付公钥ID说明](https://pay.weixin.qq.com/doc/v3/merchant/4013038816#Q%3A-%E5%A6%82%E4%BD%95%E8%8E%B7%E5%8F%96%E5%BE%AE%E4%BF%A1%E6%94%AF%E4%BB%98%E5%85%AC%E9%92%A5ID%3F)以及[微信支付公钥加密敏感信息指引](https://pay.weixin.qq.com/doc/v3/merchant/4013053257)，也可以使用微信支付平台证书公钥加密，参考[获取平台证书序列号](https://pay.weixin.qq.com/doc/v3/merchant/4013053294)、[平台证书加密敏感信息指引](https://pay.weixin.qq.com/doc/v3/merchant/4013053264)

 mobile 　选填   string

【服务人员手机号码】需更新的服务人员手机号码，不更新不需传入，该字段需要使用微信支付公钥加密（推荐），请参考[获取微信支付公钥ID说明](https://pay.weixin.qq.com/doc/v3/merchant/4013038816#Q%3A-%E5%A6%82%E4%BD%95%E8%8E%B7%E5%8F%96%E5%BE%AE%E4%BF%A1%E6%94%AF%E4%BB%98%E5%85%AC%E9%92%A5ID%3F)以及[微信支付公钥加密敏感信息指引](https://pay.weixin.qq.com/doc/v3/merchant/4013053257)，也可以使用微信支付平台证书公钥加密，参考[获取平台证书序列号](https://pay.weixin.qq.com/doc/v3/merchant/4013053294)、[平台证书加密敏感信息指引](https://pay.weixin.qq.com/doc/v3/merchant/4013053264)

 qr_code 　选填   string(1024)

**【服务人员二维码URL】**需更新的服务人员二维码，不更新不需传入 企业微信商家适用，个人微信商家不可用

 avatar 　选填   string(1024)

**【服务人员头像URL】**需更新的服务人员头像URL，不更新不需传入

 group_qrcode 　选填   string(1024)

**【群二维码URL】**需更新的服务人员所在的企业微信群的“群活码”二维码串
