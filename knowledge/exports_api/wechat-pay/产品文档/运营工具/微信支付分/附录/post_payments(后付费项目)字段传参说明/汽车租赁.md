# 汽车租赁

路径：产品文档/运营工具/微信支付分/附录/post_payments(后付费项目)字段传参说明/汽车租赁

## 1. 概述

接入微信支付分的商户需在创建、完结、修改订单金额时，通过API接口中的post_payments参数回传详细的后付费项目明细。该明细将在支付分订单详情页面展示，减少用户对订单金额的疑问。

post_payments为数组类型参数，该参数可包含多条“后付费项目明细”，至少传1条明细，最多可传100条明细，每条“后付费项目明细”包含name、amount、description、count四个参数。

## 2. 后付费项目参数传值要求

### 2.1 创建支付分订单接口

汽车租赁行业的商户在[创建支付分订单](https://pay.weixin.qq.com/doc/v3/merchant/4012587900)时，需通过post_payments【后付费项目】参数回传免押租车规则，以便于在支付分订单明细中向用户展示。

post_payments参数传值要求：

| 字段简称 | 字段名 | 类型[长度限制] | 是否必填 | 示例值 | 描述 | 传值示例 |
| --- | --- | --- | --- | --- | --- | --- |
| 后付费项目名称 | name | string [1, 20] | 必填 | 免押租车规则 | 付费项目名称，请填写“免押租车规则”六个字 | "post_payments" :  [{	"name" :   "免押租车规则",	"description" : "还车后30天内按费用标准核查车损、违章等费用，按实际费用扣费"}] |
| 后付费项目金额 | amount | int | 不填 | - | 请勿传该字段 |  |
| 后付费项目说明 | description | string [1, 30] | 必填 | 还车后30天内按费用标准核查车损、违章等费用，按实际费用扣费 | 请填写计费规则的详细说明 |  |
| 后付费项目的数量 | count | int | 不填 | - | 请勿传该字段 |  |

### 2.2 完结支付分订单接口

当服务结束后，商户调用[完结支付分订单](https://pay.weixin.qq.com/doc/v3/merchant/4012587955)接口，需通过post_payments【后付费项目】参数回传订单实际的付费项目，在汽车租赁行业中，只允许回传“租车相关费用、延迟还车费用、加油费用、ETC高速费用、车损费用、折旧费用、停运费用、闯红灯违章”这8类付费项目。

若订单无收款费用，post_payments只能有一项“租车相关费用”的后付费项目明细；

若订单有收款费用，实际收费中有“租车相关费用、延迟还车费用、加油费用、ETC高速费用、车损费用、折旧费用、停运费用、闯红灯违章”的费用，请按照实际情况填写后付费项目明细，并至少有其中一项。

例：如果订单无收款费用(0元完结订单)，传参时只能传“租车相关费用”这一项后付费项目明细，如果订单有收款费用(非0元完结)，可以传“延迟还车费用”、“ETC高速费用”等其他后付费项目明细。

post_payments参数传值要求：

| 字段简称 | 字段名 | 类型[长度限制] | 是否必填 | 示例值 | 描述 | 传值示例（无收款费用） | 传值示例（有收款费用） |
| --- | --- | --- | --- | --- | --- | --- | --- |
| 后付费项目名称 | name | string[1, 20] | 必填 | 租车相关费用 | 付费项目名称，只可填写如下取值可选取值：    租车相关费用、延迟还车费用、加油费用、ETC高速费用、车损费用、折旧费用、停运费用、闯红灯违章 | "post_payments" :  [{	"name" :   "租车相关费用",	"description" : "本次无扣款"}] | "post_payments" :  [{	"name" :   "延迟还车费用",	"amount": 1000,	"description" : "延迟还车2小时",	"count": 1}， {	"name" :   "ETC高速费用",	"amount": 2000,	"description" : "2024年9月1日在深圳高速出口产生高速费用",	"count": 1}] |
| 后付费项目金额 | amount | int | 选填 | 1200 | 付费项目金额，单位为分订单0元完结时，请勿传该字段，非0元完结时必传该字段 |  |  |
| 后付费项目说明 | description | string [1, 30] | 必传 | 骑行费用1.5元/30分钟 | 对付费项目的详细说明订单0元完结时，只能填写“本次无扣款”五个字 |  |  |
| 后付费项目的数量 | count | int | 选填 | - | 订单0元完结时，请勿传该字段，非0元完结时必传该字段 |  |  |

### 2.3 修改订单金额接口

完结支付分订单后，且订单还处于“待支付（USER_PAYING）”状态时，可调用[修改订单金额](https://pay.weixin.qq.com/doc/v3/merchant/4012587957)接口“下调”订单的金额，传值要求与完结支付分订单接口传值要求一致。
