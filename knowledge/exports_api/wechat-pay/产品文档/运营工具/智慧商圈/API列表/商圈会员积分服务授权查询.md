# 商圈会员积分服务授权查询

路径：产品文档/运营工具/智慧商圈/API列表/商圈会员积分服务授权查询

通过此API，商圈商户/服务商可自行查询用户的商圈会员积分服务授权状态。

- 接口频率：150QPS
- 前置条件：商圈商户/服务商已开通支付即积分产品功能

## 接口说明

支持商户：【普通商户】

请求方式：【GET】`/v3/businesscircle/user-authorizations/{openid}`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点 ，指引[点击查看](https://pay.weixin.qq.com/doc/v3/merchant/4012075113)

## 请求参数

Header   HTTP头参数

 Authorization 　必填　string

请参考[签名认证](https://pay.weixin.qq.com/doc/v3/merchant/4012365342#%E8%AF%B7%E6%B1%82%E6%8E%A5%E5%8F%A3%E5%90%8E%E7%AB%AF%E7%AD%BE%E5%90%8D)生成认证信息

 Accept 　必填　string

请设置为`application/json`

path  路径参数

 openid 　必填   string(64)

**【顾客OpenID】**顾客授权时使用的小程序上的OpenID

query  查询参数

 appid 　必填   string(128)

**【小程序AppID】**顾客授权积分时使用的小程序的AppID

## 应答参数

| 200 OK |  | 200 OK |
| --- | --- | --- |
|  | 200 OK |  |

 openid 　必填   string(64)

**【顾客OpenID】**顾客授权时使用的小程序上的OpenID

 authorize_state 　必填   string

**【授权状态】**顾客授权商圈积分结果，1."UNAUTHORIZED":未授权 ，2."AUTHORIZED":已授权 ，3."DEAUTHORIZED":已取消授权

可选取值：

- `UNAUTHORIZED`:   顾客无授权商圈进行支付即积分的记录
- `AUTHORIZED`:   顾客已授权商圈进行支付即积分的能力
- `DEAUTHORIZED`:   顾客已取消授权商圈进行支付即积分的能力

 authorize_time 　选填   string(32)

**【授权时间】**顾客成功授权商圈积分的时间

 deauthorize_time 　选填   string(32)

**【取消授权时间】**顾客关闭授权商圈积分的时间
