# 解冻剩余资金

路径：产品文档/扩展工具/分账/API列表/解冻剩余资金

不需要进行分账的订单，可直接调用本接口将订单的金额全部解冻给本商户

1. 调用分账接口后，需要解冻剩余资金时，调用本接口将剩余的分账金额全部解冻给本商户
2. 此接口采用异步处理模式，即在接收到商户请求后，优先受理请求再异步处理，最终的分账结果可以通过查询分账接口获取

## 接口说明

支持商户：【普通商户】

请求方式：【POST】`/v3/profitsharing/orders/unfreeze`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点 ，指引[点击查看](https://pay.weixin.qq.com/doc/v3/merchant/4012075113)
