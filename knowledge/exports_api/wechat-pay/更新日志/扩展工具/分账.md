# 分账

路径：更新日志/扩展工具/分账

## 2022-07

1.【分账】 更新《[查询分账结果](https://pay.weixin.qq.com/doc/v3/merchant/4012525210)》返回参数字段fail_reason：新增枚举值INVALID_REQUEST

2.【分账】 更新《[查询分账回退结果](https://pay.weixin.qq.com/doc/v3/merchant/4012526279)》返回参数字段fail_reason：新增枚举值INVALID_REQUEST

## 2022-04

1.【分账】 更新《[请求分账](https://pay.weixin.qq.com/doc/v3/merchant/4012524936)》、《[查询分账结果](https://pay.weixin.qq.com/doc/v3/merchant/4012525210)》fail_reason字段新增枚举：PAYER_ACCOUNT_ABNORMAL：分出方账户异常

2.【分账】 更新《[请求分账回退](https://pay.weixin.qq.com/doc/v3/merchant/4012525287)》《[查询分账回退结果](https://pay.weixin.qq.com/doc/v3/merchant/4012526279)》fail_reason字段新增枚举：PAYER_ACCOUNT_ABNORMAL：原分账分出方账户异常；原有的ACCOUNT_ABNORMAL枚举中文描述修改为“原分账接收方账户异常”

## 2022-02

1.【分账】 更新《[请求分账](https://pay.weixin.qq.com/doc/v3/merchant/4012524936)》《[查询分账结果](https://pay.weixin.qq.com/doc/v3/merchant/4012525210) 》《[解冻剩余资金](https://pay.weixin.qq.com/doc/v3/merchant/4012526374)》对参数分账失败原因（fail_reason）的描述内容以及枚举值进行了调整

## 2021-11

1.【分账】 更新《[请求分账](https://pay.weixin.qq.com/doc/v3/merchant/4012524936)》单笔订单调整为最多发起50次分账请求，每次请求最多分给50个接收方

## 2021-07

2.【分账】 更新 《[请求分账](https://pay.weixin.qq.com/doc/v3/merchant/4012524936)》、《[查询分账结果](https://pay.weixin.qq.com/doc/v3/merchant/4012525210)》、《[解冻剩余资金](https://pay.weixin.qq.com/doc/v3/merchant/4012526374)》回参分账接收方列表 新增分账明细单号（detail_id）参数
