# 商家券

路径：更新日志/运营工具/商家券

## 2021-12

1.【商家券】更新《[创建商家券](https://pay.weixin.qq.com/doc/v3/merchant/4012534855)》、《《[查询商家券批次详情](https://pay.weixin.qq.com/doc/v3/merchant/4012534928)》、《[修改商家券基本信息](https://pay.weixin.qq.com/doc/v3/merchant/4012535020)》，券详情图片（coupon_image_url）和视频号封面图（finder_video_cover_image_url）字段的图片尺寸改为：1074像素（宽）*603像素（高）

## 2021-10

1.【商家券】更新 《[创建商家券](https://pay.weixin.qq.com/doc/v3/merchant/4012534855)》，样式信息（display_pattern_info）中：
商户logo（merchant_logo_url）修改规则：若券归属商户号有认证品牌，则系统将自动拉取对应品牌logo；若券归属商户号不在认证品牌下，需自定义上传logo；
商户名称（merchant_name）修改规则：不支持商户自定义。若券归属商户号有认证品牌，系统将自动拉取认证品牌号下的品牌名称；若券归属商户号不在认证品牌下，则拉取本商户号的商户简称。

2.【商家券】 更新《[创建商家券](https://pay.weixin.qq.com/doc/v3/merchant/4012534855)》、《《[查询商家券批次详情](https://pay.weixin.qq.com/doc/v3/merchant/4012534928)》、《[根据过滤条件查询用户的券](https://pay.weixin.qq.com/doc/v3/merchant/4012534963)》、《[查询用户券详情](https://pay.weixin.qq.com/doc/v3/merchant/4012535011)》、《[修改商家券基本信息](https://pay.weixin.qq.com/doc/v3/merchant/4012535020)》，样式信息（display_pattern_info）新增视频号相关信息（finder_info）字段

## 2021-09

1.【商家券】更新 《[修改商家券基本信息](https://pay.weixin.qq.com/doc/v3/merchant/4012535020)》去除参数：商家券批次名称（stock_name）、商户logo（merchant_logo_url）、商户名称（merchant_name）
