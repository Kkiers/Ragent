# 调起支付报错：支付验证签名失败

路径：通用规则/开发须知/常见问题/调起支付报错：支付验证签名失败

1、请确认prepay_id的格式是否正确

app调起支付的为
![](https://gtimg.wechatpay.cn/resource/xres/img/202409/5236a3a065efe1276d29855d7cad26f8_16x16.svg)
```
1wx201410272009395522657a690389285100
```

而jsapi调起支付和小程序的为
![](https://gtimg.wechatpay.cn/resource/xres/img/202409/5236a3a065efe1276d29855d7cad26f8_16x16.svg)
```
1prepay_id=wx201410272009395522657a690389285100
```

2、调起支付与下单计算签名使用的商户API证书不一致

请确保调起支付与下单计算签名使用的商户API证书是同一份
